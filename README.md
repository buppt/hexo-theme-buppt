### 使用
hexo的使用方法下次补上，这里先默认大家会用=-=

把buppt文件夹拷贝到themes文件中，把_config.yml中的theme改成buppt。

修改buppt文件中的_config.yml文件，将csdn、github、微博地址换成自己的地址，author、name、introduce改成自己的信息，头像替换source/img/photohead.png即可。
